# TrendKard

A crypto-native, deck-based prediction game on Solana. Users mint NFT cards representing crypto/AI narratives, build 5-card decks, and compete in daily rounds to predict which trends will dominate. Rewards are paid automatically in SOL.

## Tech Stack

- **Chain**: Solana
- **Programs**: Anchor
- **NFTs**: Metaplex
- **Frontend**: Next.js + Tailwind
- **Oracles**: Pyth (prices), custom API (X mentions, on-chain metrics)

## Project Structure

- `programs/` - Solana Anchor program (Rust)
- `app/client/` - Next.js frontend
- `tests/` - Program tests
